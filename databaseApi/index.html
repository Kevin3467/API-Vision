Vision API

bem vindo a pagina inicial da API do Sitema Vision

essa API foi criada para o tratamento de dados backend de forma rapida e segura para resolver problemas de dados comuns na Projeta. para acessar os Cruds, vc pode utilizar os seguintes padrões de URL com as respectivas requisições:

'/create/{tabela}' (POST) para criar rows de acordo com a tabela selecionada
'/read/{tabela}' (GET) para querys completos das tabelas
'/{tabela}(no singular)/id' (GET) para querys de dados especificos em tabelas especificas
'/{tabela}(no singular)/id' (PUT) para substituir dados de rows de acordo com o id
'/{tabela}(no singular)/id' (DELETE) para apagar rows de acordo com o id

em breve mais urls dinamicas para a API
